# 项目部署指南

## 环境要求

- 微信开发者工具（最新版）
- 微信小程序账号（已认证）
- 微信云开发环境（已开通）

## 一、导入项目到微信开发者工具

1. 打开微信开发者工具
2. 选择「导入项目」
3. 项目目录选择：`/Users/a2109164/Projects/dogFood/Project7`
4. AppID 填写你的小程序 AppID（或使用测试号）
5. 点击「导入」

## 二、配置云开发环境

### 1. 开通云开发

1. 在微信开发者工具中，点击工具栏的「云开发」按钮
2. 如果尚未开通，按提示开通云开发服务
3. 选择基础版（免费）或付费版
4. 创建云开发环境，记住环境 ID

### 2. 修改云开发环境配置

修改 `/Project7/app.js` 文件中的云环境配置：

```javascript
wx.cloud.init({
  env: 'your-cloud-env-id',  // 替换为你的云环境ID
  traceUser: true,
});
```

## 三、创建云数据库集合

在云开发控制台 -> 数据库中，创建以下集合：

1. **users** - 用户表
2. **news** - 资讯表  
3. **collections** - 收藏表

详细的字段说明和权限配置请参考 `database/init-db.md`

## 四、上传并部署云函数

### 方法一：右键上传（推荐）

1. 在微信开发者工具中，右键点击 `cloudfunctions` 目录
2. 选择「同步云函数列表」
3. 然后对每个云函数目录（如 `login`、`getNewsList` 等）：
   - 右键点击云函数文件夹
   - 选择「上传并部署：云端安装依赖」

### 方法二：使用终端上传

1. 安装 wx-server-sdk（本地开发调试用）
2. 在云函数目录下安装依赖：
```bash
cd cloudfunctions/login
npm install
```
3. 然后在开发者工具中右键上传

### 需要部署的云函数列表：

| 云函数名称 | 功能说明 |
|-----------|---------|
| login | 用户登录，获取openid |
| updateUserInfo | 更新用户信息 |
| getNewsList | 获取资讯列表 |
| getNewsDetail | 获取资讯详情 |
| toggleCollection | 切换收藏状态 |
| getCollections | 获取用户收藏列表 |

3. 注意：首次运行时需要先部署云函数，否则云函数调用会失败

## 八、常见问题

### Q1: 云函数调用失败？

A: 检查以下几点：
- 云函数是否已上传并部署
- 云环境ID是否正确配置
- 云函数是否有安装依赖

### Q2: 数据库查询无数据？

A: 检查以下几点：
- 集合是否已创建
- 集合中是否有数据
- 数据库权限是否正确配置

### Q3: 提示"云函数不存在"？

A: 需要在开发者工具中右键点击云函数目录，选择「上传并部署」。

### Q4: 真机调试时无法调用云函数？

A: 检查以下几点：
- 小程序是否已上线或处于开发版/体验版
- 是否在合法域名配置中添加了云开发域名
- 云环境是否处于正常状态
